== Description ==

Find existing tags in your new post content. If found, automatically add those tags to the post whenever you save the post. For more information, visit <a href="http://dekisugi.net/wpplugins/auto-add-tags">Support</a> page.


== Installation ==

1. Upload `auto-add-tags.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it.
